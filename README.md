# Conversor de Imagens para PDF (Windows)

Aplicativo simples, leve e totalmente offline para converter imagens em arquivos PDF no Windows.

---

## 📥 Download

A versão mais recente do conversor está disponível para download no link abaixo:

👉 https://www.mediafire.com/file/ynrdt93wo4i9efp/conversor.zip/file

📦 O arquivo ZIP contém:
- Executável do conversor
- Arquivo README com instruções
- Pasta `entrada`
- Pasta `saida`

⚠️ **Aviso do Windows (SmartScreen)**  
Ao executar o programa pela primeira vez, o Windows pode exibir um aviso de segurança informando que o aplicativo não é reconhecido.  
Isso acontece porque o programa não possui assinatura digital e **não representa risco**.

Para executar:
1. Clique em **Mais informações**
2. Clique em **Executar assim mesmo**

---

## 📝 Instruções de Uso

1. Coloque as imagens que deseja converter na pasta `entrada`
2. Execute o programa
3. Os arquivos PDF serão gerados automaticamente na pasta `saida`

---

## 🖼️ Formatos suportados

JPG, JPEG, PNG, BMP, GIF, TIFF, JFIF
