# 🖼️ Conversor de Imagens para PDF (Windows)

Aplicativo simples, leve e **100% offline** para converter imagens em arquivos PDF no Windows.

---

## 🚀 Funcionalidades

* Conversão rápida de imagens para PDF
* Interface simples e fácil de usar
* Funciona totalmente offline
* Suporte a múltiplos formatos

---

## 🖼️ Formatos suportados

JPG, JPEG, PNG, BMP, GIF, TIFF, JFIF

---

## 📥 Download

👉 https://www.mediafire.com/file/ynrdt93wo4i9efp/conversor.zip/file

O arquivo ZIP contém:

* Executável do conversor
* Arquivo README com instruções
* Pasta `entrada`
* Pasta `saida`

---

## 📝 Como usar

1. Coloque as imagens na pasta `entrada`
2. Execute o programa
3. Os PDFs serão gerados automaticamente na pasta `saida`

---

## ⚠️ Aviso de Segurança (Windows)

Ao executar o programa pela primeira vez, o Windows pode exibir um aviso de segurança.

Isso acontece porque o aplicativo não possui assinatura digital.

Para continuar:

1. Clique em **Mais informações**
2. Clique em **Executar assim mesmo**

---

## 🛠️ Tecnologias utilizadas

* Python
* Biblioteca Pillow (manipulação de imagens)
* Tkinter (interface gráfica)

---

## 💻 Executar pelo código (modo desenvolvedor)

```bash
pip install pillow
python main.py
```
